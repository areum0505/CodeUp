## 1716 : 10월 30일

입력으로 월과 일이 입력되면, 혜인공주의 생일이 몇 일 남았는지 D-DAY를 계산하는 프로그램을 작성하시오.

``` c
#include <stdio.h>

int main() {
	int month, day;
	int total = 0;
	scanf("%d %d", &month, &day);

	for (int i = month; i <= 10; i++) {
		if (i % 2 == 1) {
			total += 31;
		} else {
			total += 30;
		}
	}
	total -= day;

	printf("%d", total);
}
```
