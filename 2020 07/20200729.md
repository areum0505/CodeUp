## 2621 : 약수의 합

한 정수 N을 입력받아서 N의 모든 약수의 합을 구하는 프로그램을 작성하시오.

``` c
#include <stdio.h>

int main() {
	int n, sum = 0;
	
	scanf("%d", &n);

	for (int i = 1; i <= n; i++) {
		if (n % i == 0) {
			sum += i;
		}
	}

	printf("%d", sum);
}
```

## 2622 : 최대 최솟값 구하기

반복문을 활용한 대표적인 알고리즘이 최대값과 최소값을 찾는 문제이다.

``` c
#include <stdio.h>

int main() {
	int n;
	int s[10000];
	int max = 0, min = 32767;
	int a = 0, b = 0;

	scanf("%d", &n);
	for (int i = 0; i < n; i++) {
		scanf("%d", &s[i]);
	}

	for (int i = 0; i < n; i++) {
		if (s[i] > max) {
			max = s[i];
			a = i + 1;
		}
		else if (s[i] < min) {
			min = s[i];
			b = i + 1;
		}
	}

	printf("%d : %d\n", a, max);
	printf("%d : %d", b, min);
}
```

## 2623 : 최대공약수 구하기

두 정수 a, b를 입력받아서, a, b의 최대공약수를 출력하시오.

``` c
#include <stdio.h>

int gcd(int a, int b) {
    int mod = a % b;
    while (mod > 0) {
        a = b;
        b = mod;
        mod = a % b;
    }
    return b;
}


int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", gcd(a, b));
}
```
