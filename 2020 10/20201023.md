# 1001 : [기초-출력] 출력하기01(설명)

# 1002 : [기초-출력] 출력하기02(설명)

다음 문장을 출력해보자.

Hello World
(대소문자에 주의한다.)

``` c++
#include <iostream>
using namespace std;

int main() {
  cout << "Hello World" << endl;

  return 0;
}
```

# 1003 : [기초-출력] 출력하기03(설명)

이번에는 줄을 바꿔 출력하는 출력문을 연습해보자.
다음과 같이 줄을 바꿔 출력해야 한다.

Hello
World
(두 줄에 걸쳐 줄을 바꿔 출력)

``` c++
#include <iostream>
using namespace std;

int main() {
  cout << "Hello\nWorld" << endl;

  return 0;
}
```
