# 1020 : [기초-입출력] 주민번호 입력받아 형태 바꿔 출력하기

앞의 6자리는 생년월일(yymmdd)이고 뒤 7자리는 성별, 지역, 오류검출코드이다.
주민번호를 입력받아 형태를 바꿔 출력해보자.

``` c++
#include<stdio.h>
int main()
{
    int a, b;
    scanf("%d-%d", &a, &b);
    printf("%06d%07d", a, b);
    return 0;
}
```

# 1021 : [기초-입출력] 단어 1개 입력받아 그대로 출력하기(설명) 

1개의 단어를 입력받아 그대로 출력해보자.

``` c++
#include <iostream>
#include <string>
using namespace std;

int main() {
  string s;
  cin >> s;
  cout << s;
}
```
