# 1037 : [기초-출력변환] 정수 입력받아 아스키 문자로 출력하기

10진 정수 1개를 입력받아 아스키 문자로 출력해보자.
단, 0 ~ 255 범위의 정수만 입력된다.

``` c++
#include <iostream>
using namespace std;

int main() {
  int n;
  cin >> n;
  cout << char(n);
}

```

# 1038 : [기초-산술연산] 정수 2개 입력받아 합 출력하기1(설명)

정수 2개를 입력받아 합을 출력하는 프로그램을 작성해보자.

```c++
#include <iostream>
using namespace std;

int main() {
  long long int a, b;
  cin >> a >> b;
  cout << a + b;
}
```
