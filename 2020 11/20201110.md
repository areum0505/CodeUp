1045 : [기초-산술연산] 정수 2개 입력받아 자동 계산하기

정수 2개(a, b)를 입력받아 합, 차, 곱, 몫, 나머지, 나눈 값을 자동으로 계산해보자.

``` c
#include <stdio.h>
int main()
{
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d\n", a+b);
    printf("%d\n", a-b);
    printf("%d\n", a*b);
    printf("%d\n", a/b);
    printf("%d\n", a%b);
    printf("%.2lf", (double)a/b);
    return 0;
}

```

## 1046 : [기초-산술연산] 정수 3개 입력받아 합과 평균 출력하기

정수 3개를 입력받아 합과 평균을 출력해보자.

``` c
#include <stdio.h>
int main()
{
    int a, b, c;
    scanf("%d%d%d", &a, &b, &c);
    printf("%d\n", a+b+c);
    printf("%.1lf", (a+b+c)/3.0);
    return 0;
}
```
