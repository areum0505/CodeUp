# 1101 : Hello, World!

프로그래밍 세계에 오신걸 환영합니다.

첫 번째 문제는 인사말을 출력하는 문제입니다.

기본 출력 명령은 printf입니다.

이 출력 명령으로 Hello, World!를 출력하시오.

``` c++
#include <stdio.h>
int main() 
{
 	printf("Hello, World!");
	return 0;
}
```
