# 1110 : 정수 그대로 출력하기

입력받은 정수를 그대로 출력한다.

```c++
#include <iostream>
using namespace std;  

int main() 
{
	int a;
	cin >> a;
 	cout << a;
   
 	return 0;
}
```

# 1111 : %출력

어떤 정수가 입력되면 %를 붙여 출력하시오.

```c++
#include <iostream>
using namespace std;  

int main() 
{
	int a;
	cin >> a;
 	cout << a << "%";

 	return 0;
}
```
